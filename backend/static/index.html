Acerca del Determinante

El determinante es un valor escalar calculado a partir de los elementos de una matriz cuadrada.

Se utiliza para:

  • Determinar si una matriz tiene inversa (det ≠ 0)
  • Calcular la inversa de una matriz
  • Resolver sistemas de ecuaciones (regla de Cramer)
  • Determinar si vectores son linealmente independientes

Acerca de la Inversa

La matriz inversa (A-1) es aquella que, multiplicada por la matriz original (A), resulta en la matriz identidad (I).

Condiciones para la existencia de la inversa:

  • La matriz debe ser cuadrada.
  • El determinante de la matriz debe ser distinto de cero (det(A) ≠ 0).

Método: Eliminación Gauss-Jordan

Se aumenta la matriz original con la matriz identidad [A|I] y se aplican operaciones elementales de fila hasta obtener [I|A-1].

Eliminación Gaussiana para Ax=b

Este método resuelve sistemas de ecuaciones lineales transformando la matriz aumentada [A|b] en una forma escalonada por filas.

Pasos Clave:

  • Formar la matriz aumentada [A|b].
  • Aplicar operaciones elementales de fila para triangularizar A.
  • Realizar sustitución hacia atrás para encontrar la solución x.

Proceso:

Se identifican pivotes, se eliminan elementos debajo de los pivotes y se verifica la consistencia del sistema (solución única, múltiple o sin solución).

Eliminación Gauss-Jordan

Este método resuelve sistemas de ecuaciones lineales Ax=b transformando la matriz aumentada [A|b] en su forma escalonada reducida por filas (RREF).

Características:

  • Transforma cada pivote a 1 y elimina elementos arriba y debajo de cada pivote.
  • Determina si el sistema tiene solución única, infinitas soluciones o no tiene solución.
  • Funciona con matrices cuadradas y rectangulares.

Resultado:

Se obtiene la forma escalonada reducida (RREF) de [A|b], que revela directamente las soluciones del sistema sin necesidad de sustitución hacia atrás.

Matriz A para LU

Factorización LU (Doolittle)

La factorización LU descompone una matriz cuadrada A en el producto de dos matrices: una matriz triangular inferior L (con 1s en la diagonal) y una matriz triangular superior U.

A = LU

Utilidad:

  • Resolver sistemas de ecuaciones lineales (Ax=b) de forma eficiente.
  • Calcular determinantes (det(A) = det(L) * det(U) = det(U)).
  • Calcular la inversa de una matriz.

Método: Doolittle (sin pivoteo)

Se calculan iterativamente los elementos de L y U. Esta implementación no realiza pivoteo, por lo que puede fallar si un pivote Ukk es cero.
